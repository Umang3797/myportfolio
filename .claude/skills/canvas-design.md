# Canvas Design Skill

## Core Approach

### Phase 1 - Design Philosophy
Create an aesthetic manifesto (4-6 paragraphs) establishing visual principles around:
- Form and structure
- Space and composition
- Color theory and palette
- Overall artistic direction

The philosophy should emphasize meticulously crafted work appearing as though created by someone at the absolute top of their field.

### Phase 2 - Canvas Expression
Translate the philosophy into a visual artifact that is:
- **90% visual design, 10% essential text**
- Minimal typography functioning as visual accent
- Expert spatial composition
- Museum or magazine-quality craftsmanship

## Key Principles

- **Visual-first communication**: Ideas expressed through design elements, not written explanation
- **Sophisticated craftsmanship**: Painstaking attention to detail in every element
- **Minimal text integration**: Typography as visual accent, never explanatory blocks
- **Subtle conceptual depth**: References feel intuitive without explicit announcement
- **Spatial precision**: Every element contained within boundaries with proper margins

## Execution Standards

- Flawless composition, color choices, and typography
- Polish existing components rather than adding more elements
- Cohesive artistic excellence over feature quantity
- Original designs (avoid copyright concerns)
- Treat all outputs as sophisticated art objects
