# UI/UX Pro Max - Design Intelligence

## Core Assets
- **67 UI Styles**: Glassmorphism, Claymorphism, Neumorphism, Brutalism, Dark Mode variants, Swiss Minimalism, Neubrutalism, Bento Grids, Y2K, Cyberpunk, and more
- **161 Color Palettes**: Aligned with industry-specific product categories
- **57 Font Pairings**: Google Fonts integration with variable font support
- **161 Reasoning Rules**: Industry-specific design system generation
- **99 UX Guidelines**: Structured by priority tier
- **25 Chart Types**: With accessibility and implementation guidance

## 10 Priority Tiers (Decision Framework)

1. **Accessibility** (CRITICAL) - Contrast 4.5:1, Alt text, Keyboard nav, ARIA labels
2. **Touch & Interaction** (CRITICAL) - 44x44px min targets, 8px+ spacing
3. **Performance** (HIGH) - WebP/AVIF, lazy loading, CLS < 0.1
4. **Style Selection** (HIGH) - Match style to product type, consistency
5. **Layout & Responsive** (HIGH) - Mobile-first, no horizontal scroll
6. **Typography & Color** (MEDIUM) - 16px base, 1.5 line-height, semantic tokens
7. **Animation** (MEDIUM) - 150-300ms duration, motion conveys meaning
8. **Forms & Feedback** (MEDIUM) - Visible labels, errors near fields, progressive disclosure
9. **Navigation Patterns** (HIGH) - Bottom nav <=5 items, deep linking support
10. **Charts & Data** (LOW) - Legends, tooltips, accessible color pairs

## When to Apply

Any task affecting how a feature **looks, feels, moves, or is interacted with**.

## Usage Workflow

1. Analyze requirements (product type, audience, style keywords)
2. Select appropriate UI style from 67+ options
3. Match color palette from 161 industry-specific options
4. Choose font pairing from 57 options
5. Apply reasoning rules for the specific industry
6. Validate against UX guidelines and priority tiers

## Quick Reference Checklist

- [ ] Professional UI avoids emoji as icons
- [ ] Respects safe areas on all devices
- [ ] Maintains consistent spacing rhythm
- [ ] Ensures accessibility in both light and dark modes
- [ ] Animation timing: 150-600ms range
- [ ] Color hierarchies: green=success, red=alerts, blue=trust
- [ ] Avoid: excessive decoration, poor a11y, AI purple/pink gradients

## Anti-Patterns (Always Avoid)

- Excessive decoration over function
- Poor accessibility compliance
- "AI purple/pink gradients" misuse
- Inconsistent spacing and typography
- Missing keyboard navigation
- Low contrast ratios
- Horizontal scroll on mobile
