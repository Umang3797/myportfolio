# Theme Factory Skill

Provides curated professional font and color themes with carefully selected color palettes and font pairings.

## Available Themes

1. **Ocean Depths** - Professional and calming maritime theme
2. **Sunset Boulevard** - Warm and vibrant sunset colors
3. **Forest Canopy** - Natural and grounded earth tones
4. **Modern Minimalist** - Clean and contemporary grayscale
5. **Golden Hour** - Rich and warm autumnal palette
6. **Arctic Frost** - Cool and crisp winter-inspired theme
7. **Desert Rose** - Soft and sophisticated dusty tones
8. **Tech Innovation** - Bold and modern tech aesthetic
9. **Botanical Garden** - Fresh and organic garden colors
10. **Midnight Galaxy** - Dramatic and cosmic deep tones

## Each Theme Includes
- Cohesive color palette with hex codes
- Complementary font pairings for headers and body text
- Distinct visual identity suitable for different contexts

## Application Process

1. Present available themes to user
2. Get explicit theme selection
3. Apply colors and fonts consistently throughout
4. Ensure proper contrast and readability
5. Maintain visual identity across all components

## Custom Themes

When no pre-set theme fits:
- Generate a new theme based on user description
- Choose appropriate colors/fonts from description
- Name the theme descriptively
- Show for review before applying
