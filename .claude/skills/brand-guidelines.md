# Brand Guidelines Skill

## Overview

Apply consistent brand identity and style across all visual artifacts.

## Process

1. Define or receive brand colors (primary, secondary, accent)
2. Define or receive typography choices (headings, body)
3. Apply consistently throughout all artifacts
4. Ensure proper contrast and readability
5. Maintain visual identity across all components

## Smart Font Application
- Apply heading font to headings (24pt+)
- Apply body font to body text
- Automatic fallback to system fonts when custom unavailable
- Preserve readability across all systems

## Color Application
- Use CSS custom properties / variables for brand colors
- Cycle through accent colors for visual interest
- Maintain proper contrast ratios
- Apply contextually based on element type

## Implementation
- Use RGB/hex color values for precise brand matching
- Maintain color fidelity across different systems
- Preserve text hierarchy and formatting
- Smart color selection based on background context
